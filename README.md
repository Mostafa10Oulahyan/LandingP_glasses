# Lunnette Store

Premium landing page for a luxury eyewear brand — built with **Vite + React + TypeScript + Tailwind CSS** and **framer-motion**.

## Aesthetic

Editorial-luxury direction inspired by high-fashion boutique optical houses:

- Warm cream + bone background with deep ink charcoal type
- Brass-gold accents (`#B08A4A`) used sparingly as hairlines, swatches, italic words
- **Fraunces** (variable serif) for display + **Manrope** for refined body
- Asymmetric image stacks, marquee strip, serial numbers as decorative elements
- Tasteful glassmorphism on the sticky nav, hero film-card, and newsletter input
- Subtle grain overlay, gold hairline rules, hover-reveal product quick-view

## Sections

1. AnnouncementBar (rotating)
2. Sticky Navbar (glassmorphic on scroll, mobile drawer)
3. Hero (editorial split + CTAs + trust strip)
4. Marquee
5. Featured Collections
6. Best Sellers (filterable grid + quick view)
7. Why Choose Lunnette (+ trust badges)
8. Brand Story (editorial)
9. Testimonials (carousel + stats)
10. Instagram-style Gallery
11. Newsletter
12. Footer

## Run

```bash
npm install
npm run dev
```

Open `http://localhost:5173`.

## Build

